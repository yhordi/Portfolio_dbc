BootTrack is an app that allows prospective Dev Bootcamp students to make human connections in a fast and easy way. Users can find and contact current and past DBC students in order to learn more about the program from a student's point-of-view. The app is built on a Ruby on Rails stack with a thick client using MustacheJS, AJAX, HTML 5, and CSS3. It is a heavily tested app using Jasmine, Rspec, and Capybara testing frameworks.
